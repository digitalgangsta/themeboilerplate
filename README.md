# ThemeBoilerplate

# Installe
Change theme name in ```theme.xml``` and in ```registration.php```

	
# Composer
```package.jason``` change:
- Project name - no spaces
- Repository url
- Author name
- GitHub issues url
- Project home page url

# Gulp
```gulpfile.js``` change:

- Project name
- Project url
- Select outputStyle

Install Gulp ```npm install --global gulp@3.9.1```
